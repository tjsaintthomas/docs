Detailed Description

Object pools are useful for optimization purposes. They allow retrieving game objects from a pre-allocated pool, which avoids the need to instantiate them at runtime (a potentially costly operation). They are particularly useful for handling sets consisting of many small objects like particle systems or audio SFX.
