Class GameNetworkClient

This class is responsible for handling the reception of custom network messages from the game server and routing them to the appropriate local player. Single-player mode is implemented via a second local player that uses the same system as normal human players in multiplayer modes; which is specially convenient implementation-wise (as there is no special case for it).
