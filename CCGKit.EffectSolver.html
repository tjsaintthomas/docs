Class EffectSolver

This class is responsible for resolving all the effects that take place in a game. There is an effect solver on the server side and another one on the client side. The goal of this duplicity is to allow for lag-free gameplay: the logic is always evaluated locally first so that clients do not need to wait for the server to present its results on the screen. The client's game state is still always synchronized with that of the server; it just happens to be executed locally first too.
